# flatbuffers

#### 介绍
FlatBuffers 是一个跨平台的序列化库，旨在实现最大的内存效率。 它允许直接访问序列化数据，而无需先解析，同时仍然具有很好的兼容性。

#### 支持的操作系统
- Windows
- MacOS X
- Linux
- Android
- And any others with a recent C++ compiler.

#### 支持的编程语言
- C++
- C#
- C
- Dart
- Go
- Java
- JavaScript
- Lobster
- Lua
- PHP
- Python
- Rust
- TypeScript
